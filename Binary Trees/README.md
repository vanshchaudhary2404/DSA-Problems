## 🚀 Binary Tree Interview Tips

### 🔹 Core Patterns You Must Master
- DFS Traversals → Preorder, Inorder, Postorder
- BFS Traversal → Level Order (Queue based)
- Tree Height / Depth problems
- Path-based problems (Root → Leaf)
- Tree construction (Important for interviews)

---

### 🔹 Golden Rules (Very Important)
- Always think: **DFS or BFS?**
- Use recursion for most problems (clean & fast)
- Store results globally when needed (like diameter)
- Understand **return type of recursion** clearly

---

### 🔹 Most Asked Patterns in Interviews
- Height / Depth calculation
- Diameter of Tree
- Lowest Common Ancestor (LCA)
- Maximum Path Sum
- Views (Top / Bottom / Left / Right)
- Serialization & Deserialization

## 🧠 Advanced Concepts (For Top Companies)

- Morris Traversal (O(1) space)
- Tree + Graph conversion (Distance K, Infection problems)
- BFS + Indexing (Width problems)
- Recursion + DP on Trees

---

## ⚡ Common Mistakes to Avoid

- ❌ Recomputing height again and again → causes TLE  
- ❌ Not handling NULL cases  
- ❌ Wrong recursion return values  
- ❌ Mixing BFS and DFS logic

- Step 2 → Solve Depth / Diameter / Balanced  
- Step 3 → Solve Path + LCA  
- Step 4 → Solve Construction + Serialization  
- Step 5 → Move to Advanced (Distance K, Morris)
